Elementos de Bloco e de Linha

Elementos de bloco

Sempre ocupam toda a largura do elemento pai. Quando colocados em sequência ficam um embaixo do outro. Exemplo:

p

div

h1

h2

blockquote
  1. ol
  • ul

Elementos de llinha

Ocupam apenas a largura necessária para o seu conteúdo e ficam lado a lado, caso haja largura disponível. Exemplos:

a spam em strong code abbr

Mudando o comportamento

Um elemento de bloco pode comportar-se como um elemento de linha inserindo-se no CSS o display; inline;

div em linha

p em linha

Da mesma forma, um elemento em linha pode se comportar como um elemento de bloco inserindo-se no CSS o display: block;

a em bloco
